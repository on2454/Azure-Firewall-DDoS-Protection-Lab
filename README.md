# Azure Firewall & DDoS Protection Lab

## Project Overview

This hands-on lab focused on designing and securing a Microsoft Azure network environment using Azure Firewall and Azure DDoS Protection. The objective was to gain practical experience in cloud network security, traffic filtering, threat mitigation, and secure network architecture design.

The lab simulated real-world cloud security scenarios by implementing security controls that help protect Azure resources from unauthorized access, malicious traffic, and distributed denial-of-service (DDoS) attacks.

## Project Activities

### 1. Virtual Network Configuration

* Created Azure Virtual Networks (VNets)
* Configured address spaces and subnets
* Designed network segmentation for improved security
* Validated connectivity between resources

### 2. Azure Firewall Deployment

* Deployed Azure Firewall
* Configured firewall policies
* Implemented traffic filtering rules
* Monitored inbound and outbound network traffic
* Reviewed firewall security controls

### 3. Azure DDoS Protection

* Created and configured Azure DDoS Protection Plan
* Associated DDoS Protection with Azure Virtual Networks
* Reviewed DDoS attack mitigation capabilities
* Explored protection against volumetric and protocol attacks

### 4. Network Security Design

* Implemented network segmentation
* Applied security controls to reduce attack surface
* Evaluated secure cloud architecture principles
* Reviewed Zero Trust networking concepts

### 5. Security Monitoring

* Examined security events and traffic flows
* Reviewed Azure security monitoring capabilities
* Validated security control effectiveness
